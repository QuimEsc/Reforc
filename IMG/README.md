# Recursos gràfics

- `avatar-01.png` fins a `avatar-15.png`: els quinze avatars que pot triar l'alumnat.
- `avatars-sheet.png`: full gràfic original dels avatars 01–10.
- `avatars-sheet-11-15.png`: full gràfic original dels avatars 11–15.
- `insignies/`: set insígnies individuals, inclosa `repte-extra.png`, i el full gràfic font `insignies-sheet.png`.

Cada PNG individual es pot substituir sense modificar el codi si conserva el mateix nom. És recomanable mantindre una imatge quadrada i el rostre centrat.

La resta dels planetes, camins, edificis, núvols, brillantors i animacions es dibuixa amb CSS per evitar moltes descàrregues i facilitar el canvi de trimestre.
